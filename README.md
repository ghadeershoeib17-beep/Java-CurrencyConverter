# 💱 Currency Converter (Java Console)
A simple Java application that performs currency conversion through the command line.

## 🚀 How it works
- The program lists available currencies (USD, EUR, SAR, EGP).
- The user selects the source and target currencies by number.
- The user enters the amount.
- The program calculates the result using USD as a base rate.

## 🛠 Features
- Uses Scanner for user input.
- Implements basic OOP concepts (Currency class).
- Includes error handling for invalid inputs using try-catch.
